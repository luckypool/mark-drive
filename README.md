# MarkDrive

**Google Drive™ の Markdown をブラウザで美しくプレビュー・編集。サーバーにデータは一切保存しません。**

[mark-drive.com](https://mark-drive.com/)

<br />

## クイックスタート

```bash
bun install
bun run dev
```

http://localhost:8081 を開いて使えます。

> Google Drive 連携には API キーの設定が必要です。 [セットアップガイド](./docs/setup.md)
